# Sentinel Protection Website

This repository contains the website for Sentinel Protection.

## Hosting

The website is hosted on **Cloudflare Pages** and is automatically deployed when changes are pushed to this repository.

**Live URL:** [sentinelprotection.nl](https://sentinelprotection.nl)

## Making Updates

Any changes pushed to the `main` branch will automatically trigger a new deployment. Updates typically go live within a few minutes.

## Requesting Changes

For content updates or modifications to the website, please contact the developer.
